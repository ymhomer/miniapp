Music Notation Editor Help
Overview

This editor supports Jianpu (numbered musical notation) with chords for ukulele and guitar displayed above notes. Use the Mode selector to switch between inserting notes or adding chords.

Jianpu Mode Key Mappings
  • 1–7: Insert a note (1=Do, 2=Re, ..., 7=Ti).
  • 0: Insert a rest.
  • Backspace/Delete: Delete the note before the cursor.
  • Arrow Left/Right: Move cursor between notes.
  • Home/End: Move cursor to start/end of measure.
  • Arrow Up/Down: Increase/decrease octave.
  • #/b: Add sharp/flat.
  • .: Toggle dotted note.
  • -: Add tie.
  • _: Shorten duration.
  • Enter: Add a new measure.
Chord Mode (Ukulele/Guitar) Key Mappings
  • C, D, E, F, G, A, B: Add a major chord above the selected note (e.g., C for C major).
  • m: Toggle minor chord (e.g., Cm).
  • 7: Toggle dominant 7th chord (e.g., C7).
  • M: Toggle major 7th chord (e.g., Cmaj7).
  • n: Toggle minor 7th chord (e.g., Cm7).
  • u/d: Set upstroke/downstroke.
  • Backspace/Delete: Remove the chord from the selected note.
  • Arrow Left/Right: Move cursor between notes.
  • Enter: Add a new measure.
Usage Tips
  • Switch to Ukulele/Guitar mode to add chords above specific notes. Move the cursor to a note and press a chord key (e.g., C).
  • Chords appear in blue above notes and play simultaneously during playback.
  • Click a note or measure to set the cursor position.
  • Edit time signatures by clicking (e.g., 4/4).
  • Use the Score Library to import/export scores.
  • On small screens, tap the title to show/hide the toolbar.
